Instructions

This application should be quite intuitive to use but some instructions have been included for better clarity. In addition to this, a video has also been recorded (show step-by-step video) so that the user can have a visual reference as well.

Running the Deep-FIR application:

  1. Check the models.csv file so that it includes the information for the models you want to use.
  2. Go over the options in the Settings tab.
  3. Enter the URL for the Deep-FIR backend (or else leave unchanged), and click the Test API Connection to make sure that the correct address is being used.
  4. Select the model and the processor, and click the Load Model button to correctly set up the Deep-FIR backend.
  5. Go to the Super-Resolution tab to start the image restoration process.
  6. Load an image or a video by clicking the Choose file button.
  7. If a video has been chosen, use the video player to go through the file and choose the desired frame by clicking . If an image has been chosen, this step will be skipped and you can move on to cropping.
  8. To crop the frame or image, select the and drag the box around the desired area. To cancel the crop press the undo and to move to the next step click the save button .
  9. Choose the scaling factor by which you want to super-resolve (currently only x4 is available, x2 and x3 are not yet supported) and click the respective button.
  10. Once the super-resolved image appears, you can choose to compare the before and after images by clicking and you can also download the upsampled image as a PNG by clicking download button .
  11. If the before and after comparison is chosen, three images will be presented. The first image is interactive and allows you to drag the red line over the image to see how certain features have been improved. The other two just show the before and after at the same size.
  12. At the end of the image comparison, there is another download button, which allows you to save a PDF which compares the original image with the super-resolved one.
